## 快速开始

### 前置条件

- macOS
- iTerm2
- Python 3.9+
- 已启用 iTerm2 Python API Server

### 安装

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
```

### 运行

```bash
python3 -m multi_iterm2_manager
```

或使用脚本：

```bash
./start.sh
```

### 测试

当前仓库未发现现成测试文件；如需自测，建议优先：

MITERM_BACKEND=mock python3 -m multi_iterm2_manager
